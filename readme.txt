== Description ==

Frontend Developer provides an easy way for theme developers see exactly what viewport resolution they are currently viewing.
In version 2.0 Frontend Developer is now hooked into the admin toolbar!

= Other features include =
1. Displays common media query resolutions ( ie. Mobile, Tablet, Desktop )
1. Highlights defined widget areas
1. Highlights links that do not have a destination set
1. Highlights images without alt tags set

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-front-end-developer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
